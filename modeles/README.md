# Modèles de Google Sheets

Ce dossier peut contenir des modèles (Excel `.xlsx` ou CSV) réutilisables lors de la
création d'un Google Sheet (« À partir d'un modèle »).

Au runtime, l'application utilise le dossier **`%LOCALAPPDATA%\BadmintonClub\modeles`**
(inscriptible, y compris pour une version installée). C'est là que sont enregistrés les
modèles via le bouton **« ⭐ Enregistrer comme modèle »** (options d'un Sheet), et c'est
ce dossier qui s'ouvre par défaut quand on clique dans la zone de dépôt.
